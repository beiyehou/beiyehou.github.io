跨站脚本攻击(XSS)

XSS 跨站脚本攻击

XSS (Cross Site Script)，为了和 CSS 区分就把 C 改成了 X。

特征和原理

XSS 通常发生在黑客通过向用户的浏览网页中注入一些恶意 Js 片段、HTML、Flash 或者任意浏览器可能执行的内容或者代码片段，已达到窃取用户隐私数据，cookie， session 信息或者将用户引流到钓鱼网站来实施攻击。
