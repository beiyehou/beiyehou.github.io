安全世界观

安全三要素：机密性(Confidetiality)、完整性(Integrity)、可用性(Availability)

优秀的安全方案需要具备几个特点：

  1. 能够有效解决问题
  2. 用户体验好
  3. 高性能
  4. 低耦合
  5. 易于扩展与升级
